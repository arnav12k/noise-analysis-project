# Noise Analysis in Communication Channel

## Objective
To model and analyze noise in communication systems using probability distributions.

## Description
This project simulates the effect of:
- Gaussian (Normal) Noise
- Poisson Noise

on a sinusoidal signal using MATLAB.

## Output
The program generates:
- Original signal
- Signal with Gaussian noise
- Signal with Poisson noise
- Histogram comparison
- Signal-to-Noise Ratio (SNR)

## Files
- noise_analysis.m → MATLAB code
- output_graph.png → Graph results
- report.pdf → Project report

## How to Run
1. Open MATLAB
2. Run `noise_analysis.m`
3. Observe graphs and SNR values
